# Farcaster SBT Generator

A web application that generates Soulbound Tokens (SBTs) for Farcaster users based on their FID.

## Features

- Generate SBT preview with Farcaster logo and user FID
- Real-time ETH price conversion ($1 USD)
- MetaMask integration for direct payments
- Responsive design
- Copy to clipboard functionality

## Deployment

### GitHub Pages
1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Select the main branch as source

### Vercel
1. Fork this repository
2. Go to [vercel.com](https://vercel.com)
3. Import your forked repository
4. Deploy with default settings

## Usage

1. Enter your Farcaster ID (FID)
2. Click "Generate SBT" to see preview
3. Send $1 worth of ETH to the displayed address
4. Or use MetaMask for direct payment

## Technology Stack

- HTML5, CSS3, JavaScript
- Web3.js for Ethereum integration
- CoinGecko API for ETH price
- Responsive design
